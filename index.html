How it works

Set goals. The agent does the rest — no human clicks required.

1

Create the agent

Spin up a Node.js process with the Sphere SDK. The agent gets its own wallet, @nametag, and DIRECT address.

2

Fund the treasury

Send UCT to your agent's @nametag. The agent detects incoming transfers and updates its balance.

3

Set rules

Create recurring payment schedules, balance thresholds, or auto-invest targets via the dashboard.

4

Go autonomous

The agent runs a 1-hour loop checking the treasury, executing rules, and DMing you reports.
